# classical-music-generation-model
Machine learning experiment for generating classical-style music.

Windows/Linux Users, install these torch versions: 
torch==2.5.1+cu121
torchaudio==2.5.1+cu121
torchvision==0.20.1+cu121

Mac Users, install these versions instead:
torch==2.5.1
torchaudio==2.5.1
torchvision==0.20.1


Tested with Python 3.11.5
